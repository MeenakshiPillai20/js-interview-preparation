1. What is the for-in loop in JavaScript? Give its syntax?

Ans. In JavaScript the for-in loop is a basic control statement that allows you to loop through all the properties of an object. The statements of code found within the loop body will be executed on once for each property of the object.

Syntax :


      for (variable in object){
        //statements
      }
    

      const student = {
        name: 'Monica',
        class: 7,
        age: 12
      }
      for ( let key in student ) {
        console.log(key,` => `,student[key]);
      }

Output :

        name  =>  Monica
        class  =>  7
        age  =>  12

2. Explain Local Scope, Block Scope, Functional Scope and Scope Chain in javascript?

Ans. In global scope variables are declared outside of all functions are known as global variables and in a global scope. Global variables are accessible anywhere in the program.

In functional scope variables that are declared inside a function are called local variables and in a functional scope. Local variables are accessible anywhere inside the function.

Local scope it is basically similar to functional scope has local scope is defined inside functional scope but inside brackets opening and closing { }.

In block scope variable that are declared inside a specific block and can't be accessed outside of the block. In order to access the variable of that specific block we need to create object for it.

JavaScript engine uses scopes to find out the exact location or accessibility of variable that particular process is known as scope chain.

4. what is Symbol?

Ans. Symbols are introduced in ES6 version of JavaScript a new primitive data type of JavaScript like string, number, boolean, etc. Basically symbol is a building object whose constructor returns a symbol also called symbol value that guarantees to be unique and immutable ( cannot be changed ). You can create symbol by calling the symbol ( ) Global factory function. 


      let id = Symbol("id");
      let person = {
          name: "Jack",
          [id]: 123 
      };
      console.log(person);

Output :

       { name: 'Jack', [Symbol(id)]: 123 }

5. Write code to explain map and filter in arrays?

Ans. Map method is used for creating a new array from an existing one, applying a function to each one of the element of the first array. Filter method takes each element in an array and it applies a conditional statement against it. If the conditional returns true, the elements get pushed to the output array. If the condition return false, the element does not get pushed to the output array. 


      let n = [1, 2, 3, 4];
      let doubled = n.map(item => item * 2);
      console.log(doubled);

      let num = [1, 2, 3, 4];
      let evens = num.filter(item => item % 2 === 0);
      console.log(evens)

Output :

       [ 2, 4, 6, 8 ]
       [ 2, 4 ]

6. Explain passed by value and passed by reference?

Ans. In Pass by Value, function is called by directly passing the value of the variable as an argument. Basically the parameter passed as an argument creates its own copy. So any changes made inside the function is made to the copied value not to the original value.

In Pass by Reference function is called by directly passing the reference or address of the variable as an argument. Basically parameter passed as an argument does not create its own copy, it refer to an original value so changes made inside function affect the original value. In JavaScript array and object follows Pass by Reference property.

7. Please explain Self Invoking Function and its code?

Ans. Self-invoking functions also called self-executing functions these are nameless anonymous function that is invoked immediately after its definition. These are enclosed inside a set of parentheses followed by another set of parentheses ( ), which does the execution. 


      (function(x){
        console.log(x);
      })("Hello, World!");

Output :

      Hello, World!
